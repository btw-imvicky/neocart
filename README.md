# NeoCart

A modern beginner-friendly e-commerce website built using HTML, CSS, and JavaScript.

## Features

- Product listing
- Shopping cart
- Quantity controls
- Search functionality
- Dark mode
- Persistent cart using LocalStorage
- Checkout page

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Future Improvements

- Backend integration
- Payment gateway
- User authentication
- Product database
